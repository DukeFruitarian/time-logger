== Базовая реализация учёта времени работы над проектом

После авторизации пользователь может создавать проекты, над которыми будет вести работу. Работать можно только над одним проектом одновременно. После выбора проекта, ведётся учёт времени. После окончания сессии работы, время записывается в базу данных, с указанием начала и конца периода. При просмотре информации по проекту, выдаются все периоды и общее время работы над ним.

Задеплоено http://base-time-logger.heroku.com
